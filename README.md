# 프로그램 설명
## 추진 배경
```
정부의 알뜰폰 지원 등 다양한 이유로 경쟁이 과열되어 통신사 제로섬 게임이 고착화되어 이탈 고객 증가, 과금액 감소 등을 통해 `Z 통신사 매출감소`
```

## 프로그램 목적
```
`이탈률 여부 및 이탈자 이탈시기 예측 모델`, `경쟁사로 이탈자 영향인자 분석`, `과금액 감소의 원인 분석`으로 도출된 결과에 따른 `적절한 마케팅 전략`으로 `매출 증가`
```

# 프로그램 활용 방법
```
`jupyter notebook`을 통해 활용가능  
1. `C2_데이터전처리`파일 실행 후, 최종 데이터 파일 `tele_customer_final.csv` 파일 도출  
2. `C2_EDA`파일 실행을 통해, 가능성 있는 영향인자 색출을 위한 EDA 전과정 확인  
3. `C2_평균과금액`을 통해 평균과금액에 감소의 원인을 주는 인자 분석  
4. `C2_경쟁사`를 통해 경쟁사로 이탈자 원인 분석  
5. `이탈자의 이탈 여부 및 이탈 시기 학습을 통한 최종 `이탈 여부 및 이탈 시기 예측`  
i) `C2_이탈여부` 실행하여 이탈 여부 확인  
ii) `C2_이탈여부예측` 실행하여 이탈여부 예측  
iii) `C2_이탈시기예측`을 통해 ii)에서 이탈 예측한 고객 중 이탈시기 예측
```
